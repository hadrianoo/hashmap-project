# Hash Map

JavaScript implementation of a Hash Map data structure with full test coverage.

## Installation

npm install

## Run tests

npm test

## Methods

hash, set, get, has, remove, length, clear, keys, values, entries.

## Built with

JavaScript, Jest

## What I learned

- How hash functions work and why they enable O(1) data access instead of traversing the entire structure
- How to handle collisions using linked lists (chaining)
- How to adapt and reuse existing code (linked list) to meet new requirements
- How to write tests before implementation (TDD approach)
